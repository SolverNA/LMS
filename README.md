# Система управления обучением (LMS)

## Общее описание
Данный проект представляет собой веб-приложение — **Learning Management System (LMS)**, предназначенную для автоматизации образовательных процессов в высшем учебном заведении. Система обеспечивает централизованное взаимодействие между учебным офисом (деканатом), преподавательским составом и студентами, упрощая обмен учебными материалами и контроль успеваемости.

## Основной функционал по ролям

### 1. Деканат (Администратор)
* **Управление расписанием:** Формирование и редактирование учебных планов и графиков занятий.
* **Управление пользователями:** Регистрация и ведение баз данных студентов и преподавателей.
* **Информационная поддержка:** Рассылка официальных уведомлений и файлов.

### 2. Преподаватель
* **Создание учебных заданий:** Размещение описаний курсов и постановка задач с возможностью прикрепления методических материалов.
* **Проверка работ:** Рецензирование и оценка выполненных студентами заданий.
* **Коммуникация:** Обмен сообщениями и файлами со студентами в рамках учебного процесса.

### 3. Студент
* **Доступ к материалам:** Просмотр актуального расписания и списка заданий.
* **Сдача работ:** Отправка выполненных заданий в систему с прикреплением необходимых файлов.
* **Обратная связь:** Получение оценок и консультации с преподавателями через систему сообщений.

## Технологический стек
Проект реализован на современном стеке технологий, обеспечивающем высокую производительность и масштабируемость:
* **Backend:** Java, Spring Boot (Spring Security, Spring Data JPA).
* **Frontend:** React (State management, Axios для API-запросов).
* **База данных:** PostgreSQL (Хранение данных пользователей, расписания, связей заданий и вложений).
* **Обмен данными:** REST API с поддержкой передачи мультимедийных файлов.
